火火兔微信WiFi故事机内容服务器API
===
本文档提供给火火兔微信WiFi故事机的微信前端H5开发，可利用本文档提供的API展示云端故事内容、下载点播故事到玩具、遥控播放等

# 页面授权
## 请求授权
### 描述
获取或验证授权码
### URL
**`/rest/auth/request`**
### GET method
|参数名|参数类型|是否必须|描述|
|-----|:----:|:----:|:----|
| gzh_id | String | 必选 | 授权使用的微信公众号ID |
| token | String | 可选 | 页面授权token (与code二选一) |
| code | String | 可选 | 微信登录code (与token二选一) |

### Reply
返回授权信息内包含
* `token`: 用于页面后续访问的页面授权token
* `userId`: 用户ID
* `devices`: 绑定的设备ID

### Examples
```
{
  "err": 0,
  "msg": "ok",
  "token": "7gW-gG2sQQ27NBEfaMJpyQ",
  "userInfo": {
    "userId": "ot-vPwuqOlUs-C37LbLliXOliucQ",
    "devices": [
      "TT80270"
    ]
  }
}
```

## 获取设备profile
### 描述
获取当前设备的信息和支持的功能列表，随着产品类型的功能的迭代，功能项目会不断增加。使用此接口做各种型号设备、不同版本、不同功能定义的兼容
### URL
**`/rest/device/profile`**
### GET method
|参数名|参数类型|是否必须|描述|
|-----|:----:|:----:|:----|
| device_id | String | 必选 | 故事机设备ID |
| token | String | 必选 | 页面授权token |

### Reply
主要功能选项
* `logo`: 产品形象图（显示在页面底部播放条上）
* `wifiCtrl`: 支持WebSocket远程遥控
* `localPlaylist`: 0. 不支持本地歌单（G63）; 1. 支持本地歌单V1接口; 2. 支持本地歌单V2接口; 3. 支持本地歌单V3接口
* `onlineNob`: 0. 不支持点读功能；1. 支持点读功能
* `remotePoweroff`: 0. 不支持远程关机；1. 支持远程关机；2. 支持远程立即关机和定时关机；

> 注： 习惯功能目前需要按productId来判别，只有当productId为之前的WiFi故事机，且不是G6时，支持习惯提醒功能

# 遥控接口

* 通过页面与服务器建立websocket长连接，来实现遥控和设备播放状态变化通知
* WebSocket地址：**`ws://{server}/remote?deviceId={deviceId}&token={token}`**
## 上行数据包格式
```
{
  "msgId": {msgId},
  "deviceId": {deviceId},
  "message": {message}
}
```
* `msgId` 消息类型 如心跳包为"heartBeat", 播放控制为"music_play"
* `deviceId` 设备号 消息接受者 (可为空)
* `message` 消息内容 (可为空)
`JSON内包含的回车和空格可清除`

## 下行数据包格式
```
{
  "msgId": {msgId},
  "deviceId": {deviceId},
  "message": {message}
}
```
* `msgId` 消息类型 如在线状态通知为"online", 播放状态通知为"play_state"
* `deviceId` 设备号
* `message` 消息内容

## 长连接保持
页面每隔30秒发送一次心跳包，当心跳包发送失败、或页面发现WebSocket连接失效时，需要重连
心跳包格式
```
{
  "msgId": "heartBeat"
}
```
## 播放控制命令
播放控制命令格式
```
{
  "msgId": "music_play",
  "deviceId": {deviceId},
  "message": {message}
}
```

|功能|指令(message)|备注|
|-----|----|:----|
| 播放 | **play_state:on:{timestamp}** | 控制设备开始播放 |
| 暂停 | **play_state:off:{timestamp}** | 控制设备暂停播放 |
| 上一首 | **before:{timestamp}** | 控制设备播放上一首 |
| 下一首 | **next:{timestamp}** | 控制设备播放下一首 |
| 切换播放模式 | **cyclical_mode:[single\|list]:{timestamp}** | `single`: 单曲循环<br>`list`: 列表循环 |
| 播放设备播放列表 | **local:{playlist_id}:{audio_id}:{timestamp}** | 控制设备播放本地播放列表中的某一首音频<br>并从当前这个音频开始循环播放整个播放列表 |
| 播放在线播放列表 | **cloud:{playlist_id}:{audio_id}:{timestamp}** | 控制设备播放在线播放列表中的某一首音频<br>并从当前这个音频开始循环播放整个播放列表 |
| 播放USB拷贝播放列表 | **usb:{folder_name}:{file_name}:{timestamp}** | 控制设备播放USB拷贝播放列表中的某一首音频<br>并从当前这个音频开始循环播放整个播放列表  |
| 播放USB拷贝的V3播放列表 | **usb_utf8:{folder_name}:{file_name}:{timestamp}** | 控制设备播放USB拷贝播放列表中的某一首音频<br>并从当前这个音频开始循环播放整个播放列表  |
| 推送FM | **trylistOrd:{playlist_id}:{audio_id}:{timestamp}** | 推送火火兔FM节目到设备  |
| 关机 | **power_off:{timestamp}** | 立即关机 |
| 定时停止播放 | **stop_after_seconds:{seconds}:{timestamp}** | `seconds`后停止播放 |
| 取消定时停止播放 | **cancel_auto_stop:{timestamp}** | 取消自动停止 |


* `timestamp` 单位为秒

## 设备在线状态变更通知
```
{
  "msgId": "online",
  "deviceId": {deviceId},
  "message": {
    "online": {on|off}
  }
}
```

## 设备播放状态变更通知
```
{
  "msgId": "play_state",
  "deviceId": {deviceId},
  "message": {
    "timestamp": {timestamp},
    "list_id": {listId},
    "music_id": {musicId},
    "cyclical_mode": {mode},
    "play_pause": {play|pause},
    "volume": {volume}
  }
}
```

* `list_id` 播放列表ID或文件夹名称
* `music_id` 音频ID或音频文件名
* `cyclical_mode` 播放模式 single: 单曲循环 list：顺序播放
* `play_pause` 当前是否正在播放
* `volume` 当前音量


## 设备状态变更通知(新设备)
```
{
  "msgId": "all_state",
  "deviceId": {deviceId},
  "message": {
    "timestamp": {timestamp},
    "playChannel": {playChannel},
    "listId": {listId},
    "audioId": {audioId},
    "playMode": {playMode},
    "volume": {volume},
    "totalSeconds": {totalSeconds},
    "seconds": {seconds},
    "rssi": {rssi},
    "ssid": {ssid},
    "battery": {0-100},
    "charge": {on|off},
    "light": {0|1},
    "timedOffSec": {timedOffSeconds},
    "silentMode": {0|1},
    "keyLock": {0|1}
  }
}
```

* `timestamp`: 设备当前时间，尽可能保证与服务器时间同步
* `play_channel`, `playlist_id`和`audio_id`

|play_channel|名称|playlist_id|audio_id|
|-----|-----|-----|:----|
| playlist | 已添加的播放列表 | 播放列表ID | 音频ID |
| folder | 本地存储文件夹 | 文件夹名称 | 音频文件名称 |
| folder_utf8 | 本地存储文件夹UTF8版 | 文件夹名称 | 音频文件名称 |
| rplaylist | 线上试听播放列表 | 播放列表ID | 音频ID |
| push | 单曲推送 | pushId | 音频ID |
| audio | 单曲试听 | 0 | 音频ID |
| bluetooth | 蓝牙模式 | 0 | 0 |
| fm | 收音机模式 | 电台ID | 直播填0, 播放历史节目时, 填入历史节目信息 |
| viaudio | 语音点播单曲 | 0 | 音频ID，无ID时填0 |
| url | 网络音频 | 0 | 0 |

* `play_mode`: 设备当前播放模式设备当前播放模式
* `play|pause|stop`: play 播放， pause 暂停， stop 停止(播放和暂停时，需要同时带有播放内容)
* `volume`: 设备当前音量(0-100: 0为静音)
* `total_seconds`: 当前正在播放音频的总时长 (单位：秒) 
* `seconds`: 当前播放进度 (单位：秒)
* `rssi`: 当前wifi信号强度 (带符号的整数值：如-85)
* `ssid`: 当前连接的wifi SSID
* `battery_capacity`: 当前剩余电池容量 (0-100)
* `charge_state`: 当前充电状态 (on|off)
* `light_state`: 当前七彩灯状态 (1|0)
* `timed_off_seconds`: 定时停止时间(单位：秒)
* `silent_mode`: 当前是否在免打扰模式(1|0)
* `keylock`: 是否已上童锁(1|0)
